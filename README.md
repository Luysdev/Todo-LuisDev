<h1>Projeto Lista de Tarefas</h1>
<p>Este projeto teve como principal objetivo o aprendizado das tecnologias utilizadas e o treinamento para lidar com demandas semelhantes no futuro. Foi idealizado como uma aplicação simples, pensando em uma possível integração com banco de dados. Parte fundamental do projeto é o seu caráter prático como uma lista de tarefas, sendo desenvolvido de maneira a utilizar SESSIONS para manter as informações no navegador do usuário, garantindo uma experiência intuitiva e eficiente.</p>
